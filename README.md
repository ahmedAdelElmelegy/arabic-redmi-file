# Arabic Curriculum: Detailed Content for Review

This document contains all curriculum content, including vocabulary and examples, for linguistic review.

## Level: Foundations (A0) / أساسيات (A0) (A0)
**English Description:** Master the Arabic script, sounds, and survival phrases.
**Arabic Description:** أتقن الكتابة العربية، الأصوات، والعبارات الأساسية.

### Unit: Unit 1: Alphabet & Sounds / الوحدة 1: الحروف والأصوات (a0_u1)
**Description (EN/AR):** Learn the core 28 letters and their various shapes. / تعلم الحروف الـ 28 الأساسية وأشكالها المختلفة.
